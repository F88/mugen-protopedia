# URLから直接起動する機能

URLクエリパラメータを使用してアプリケーションの特定の機能を直接起動する機能についてまとめる。

## IDs を指定して開く

MUGEN Protopedia のTOP画面にあるIDを指定してプロトタイプを開く機能がある。
指定されたIDについてその機能をまとめて自動的に連続実行するというイメージ。

### Prams 仕様

- `id`: カンマ区切りのIDリスト。IDは数字で構成される。指定された順番に処理、重複も許容される。
- `title`: オプション。新たに画面上に表示するタイトル。音楽のプレイリストのタイトルに相当する。
